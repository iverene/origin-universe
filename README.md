# 🌌 The Origin of the Universe

> *"From a single spark came every star, every galaxy, every planet... and eventually, you."*

An immersive cinematic 3D web experience that takes visitors on a seamless journey through **13.8 billion years of cosmic history**. Rather than reading about the universe, users experience it firsthand by drifting through space as the universe evolves around them—from the Big Bang to the distant future.

Built with modern web technologies, the project combines real-time 3D graphics, cinematic storytelling, smooth scroll-driven animation, and atmospheric sound design to transform a website into an unforgettable interactive experience.

---

# ✨ Overview

**The Origin of the Universe** is a single-page, scroll-driven storytelling website designed to inspire curiosity and wonder.

The visitor is not controlling a spaceship or playing a game.

Instead, they become an invisible observer floating through space, witnessing the birth and evolution of the cosmos.

Every scroll represents the passage of time.

Every transition is seamless.

Every scene flows naturally into the next without cuts, page changes, or loading screens.

The experience is inspired by documentaries, IMAX films, and cinematic space epics, emphasizing emotion and visual storytelling over traditional website navigation.

# 🎨 Design Philosophy

The project follows one guiding principle:

> **Show, don't tell.**

Instead of lengthy explanations, the story is communicated through:

* Lighting
* Camera movement
* Scale
* Animation
* Environmental evolution
* Music
* Minimal cinematic subtitles

Visitors should feel as though they are experiencing the universe rather than reading about it.

---

# 🎥 Cinematic Experience

The experience is designed as one uninterrupted journey.

There are:

* No page reloads
* No scene cuts
* No traditional navigation
* No visible UI during the experience

The camera gently drifts through space while the universe evolves around the visitor.

Scrolling controls the passage of time rather than page navigation.

---

# 🎵 Audio Design

Audio evolves alongside the universe.

Examples include:

* Deep ambient drones

* Spatial environmental effects

The soundtrack complements the visuals without overpowering them.

# ⚡ Performance

To ensure a smooth experience, the project uses:

* Lazy loading
* GLB model optimization
* Draco compression
* Texture compression
* Adaptive device pixel ratio
* Instanced rendering
* Efficient particle systems

Target performance:

* 60 FPS on desktop
* Smooth experience on modern mobile devices

# 🚀 Getting Started

## Clone the repository

```bash
git clone <repository-url>
```

## Install dependencies

```bash
npm install
```

## Run the development server

```bash
npm run dev
```

Open your browser and navigate to:

```text
http://localhost:3000
```

---

# 📦 Build for Production

```bash
npm run build
```

Run the production server:

```bash
npm run start
```
